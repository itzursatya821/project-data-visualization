# Chocolate Sales Performance Analysis & Revenue Forecasting
This project analyzes chocolate sales data to identify trends, top-performing products, and regional performance using data visualization techniques. The objective is to transform raw sales data into meaningful insights through graphical representations such as bar charts, line graphs, pie charts, and scatter plots.

Using Python, Pandas, Matplotlib, and Seaborn, the project highlights sales distribution, revenue patterns, seasonal trends, and the relationship between price and sales volume. The insights generated help support data-driven decision-making and business strategy improvement.
